Webpage to understand Linux, vi and GIT commands

Welcome to my home page!!

This webpage is created to list down useful linux, vi and git commands

Linux Commands

pwd: To show present working directory

cd: To change directory

ls: to list all files in the folder

mkdir: to create a new directory/folder

rm file: To delete a file

rm -r dir: To delete directory dir

mv file1 file2: To rename/move file1 to file2. If file2 is an existing directory, moves file1 to directory file2

cp file1 file2: To copy file1 to file 2

Vi Commands

vi filename: To edit filename starting at line 1

vi -r filename: To recover filename that was being edited when system crashed

i: To get into Insert mode in vi

esc: To get out of insert mode

Shift + colon: Command prompt for vi

w: Command to write into vi

q: Command to quit vi

wq: Command to write and quit vi

Git Commands

git init: To create a repository

git add filename: To add the fileas it looks now to your next commit

git status: To show modified files in your working directory, staged for your next commit

git commit -m "message": To commit changes to a file

git checkout -b branchname: To create a new branch and checkout

git checkout branchname: To checkout branch

git push [alias] [branch]: To transmit local branch commits to the remote repository branch

git merge branchname: To merge the branch with master (Note: You have to be on the branch that needs to be merged)
